# Contributing

Thanks for your interest! This is a personal project, but ideas and PRs are welcome.

## What needs help

- **New hypotheses** — If you have ideas for agent architecture improvements, open an issue with a Hypothesis statement
- **Translations** — Chinese docs available, more languages welcome
- **Case studies** — Have you applied similar governance to your agent? Write it up!

## Process

1. Open an issue or discussion
2. For hypotheses, use the format in `architecture/hypotheses/H-XXX.json`
3. For proposals, use `architecture/proposals/AP-XXX.md`

## Note

This is a single-developer project running on Windows/Hermes Agent. Architecture decisions are driven by real usage data, not theoretical perfection.
